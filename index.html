Bienvenido al Sistema de Asistencia Escolar

Plataforma diseñada para gestionar estudiantes, controlar accesos, registrar asistencia diaria, consultar historiales y respaldar información de manera rápida, segura y eficiente.

Ingresar al Sistema

👨‍🎓 Gestión de Estudiantes

Registro, actualización y consulta de estudiantes.

🔐 Control de Acceso

Autenticación de usuarios y manejo de roles.

📋 Asistencia Escolar

Registro diario de asistencia, tardanzas y faltas.

📸 Fotografía de Estudiantes

Identificación visual para mejorar el control escolar.

📅 Historial de Asistencia

Consulta y seguimiento histórico de asistencia.

🔍 Búsquedas y Filtros

Localización rápida de estudiantes y registros.

💾 Respaldo JSON

Exportación e importación de información del sistema.

📊 Preparado para Dashboard

Base para reportes estadísticos y gráficos del Sprint 3.

Objetivo del Sistema

Facilitar el control y seguimiento de la asistencia escolar, permitiendo una gestión eficiente de estudiantes y docentes mediante herramientas tecnológicas modernas.

En el Sprint 2 se incorporan funcionalidades de registro de asistencia, historial, filtros de búsqueda, identificación mediante fotografías y respaldo de información en formato JSON.
